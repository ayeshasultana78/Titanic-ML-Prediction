# Titanic Survival Prediction 🚢🤖

## Overview
Machine Learning model to predict Titanic passenger 
survival using Decision Tree and Random Forest.

## Results
| Model | Accuracy |
|-------|----------|
| Decision Tree | 74.86% |
| Random Forest | 80.45% |

## Features Used
- Pclass (Passenger Class)
- Sex
- Age
- Fare

## Tools Used
- Python
- Pandas
- Scikit-learn

## Key Finding
Gender and Class were the strongest 
survival predictors.
